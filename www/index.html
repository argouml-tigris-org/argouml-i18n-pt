Missão

O objectivo deste sub-projecto é criar uma localização completa do ArgoUML em Português.

A localização para o Português do Brasil é realizada no sub-projecto argouml-pt-br. Para realizar a localização de outras variações do Português terá de se realizar uma proposta na lista de email dos developers do ArgoUML de modo a que se crie outro sub-projecto para o efeito.

Estado

Parte integrante do projecto principal desde a versão de desenvolvimento 0.21.3.

A tradução da aplicação está completa, embora alguns termos sejam discutíveis. A tradução ainda não foi actualizada para o Acordo Ortográfico (AO).

Existem alguns erros que estão documentados no Jenkins job do argouml-pt.
